## Run Locally

Clone the project

```bash
  git clone https://github.com/caamaann/axa-mandiri-test
```

Go to the project directory

```bash
  cd axa-mandiri-test
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
